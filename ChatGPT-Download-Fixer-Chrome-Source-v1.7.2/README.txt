ChatGPT Download Helper v1.7.2

What changed in v1.7.2:
- Fixes the missing auto-download trigger after capture
- Fixes the popup Auto-download checkbox so it loads and saves correctly
- Auto-download remains ON by default
- Adds explicit auto-download result entries to Debug mode

Install:
1. Extract this folder
2. Chrome: open chrome://extensions, enable Developer mode, and Load unpacked
3. Refresh chatgpt.com after installing

Use:
1. Open chatgpt.com
2. Click a generated file download
3. If Auto-download is enabled, the file should download automatically
4. Open the popup to confirm the toggle stays enabled
5. In Debug mode, look for an 'auto-download' log entry after each capture
